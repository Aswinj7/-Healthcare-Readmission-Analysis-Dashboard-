
# 🏥 Healthcare Readmission Analysis Dashboard

## 📌 Project Overview
This project analyzes hospital readmission patterns using healthcare data. The goal is to identify key factors influencing patient readmissions and provide actionable insights.

---

## 📊 Dataset
- 100K+ patient records
- Includes age, gender, hospital stay, medications, and readmission status

---

## 🛠️ Tools Used
- Power BI
- Excel
- DAX

---

## 📈 Key Insights
- 46% of patients were readmitted
- Higher readmission in age group 50–80
- Increased medication usage linked to higher readmission
- Longer hospital stay indicates higher risk
- Medication change has limited impact
